# Лабораторная работа — Вероятностные модели

Лабораторная работа из курса Яндекс ШАД по машинному обучению. Тема — вероятностные модели.

В качестве данных используются твиты Илона Маска (таргет — число лайков / 100) и синтетический датасет для классификации.

---

## Структура

### Часть 1. Регрессия

**1.1. Зачем нужны вероятностные модели**  
Показываем, что коэффициенты обычной линейной регрессии нестабильны — через бутстреп видно, что коэффициент при `tweet_length` меняет знак в зависимости от подвыборки.

**1.2. L1-регуляризация**  
Lasso с подбором `alpha` по кросс-валидации на логарифмической сетке (7 значений от `1e-3` до `1e+3`). Оставляем признаки с весами > `1e-10`.

**1.3. Статистическая значимость коэффициентов**  
Проверяем значимость через t-статистику и p-values с помощью `statsmodels.OLS`. Убираем признаки с p > 0.05.

**1.4. Отбор признаков через AIC**  
Жадный forward selection по критерию AIC. Отбираем 40 признаков.

**1.5. Гетероскедастичность**  
Проверяем предположения модели: scatter-plot остатков + тест Бройша-Пагана. Гетероскедастичность есть, пересчитываем стандартные ошибки с `cov_type='HC3'`.

**1.6. Пуассоновская регрессия**  
Таргет — целые числа, распределение похоже на Пуассона, а не на нормальное. Пишем `SGDPoissonRegression` с momentum'ом с нуля, затем сравниваем со `statsmodels.Poisson`.

### Часть 2. Классификация

**2.1. Калибровка**  
Реализуем `IsotonicallyCalibratedModel` — калибратор на основе изотонической регрессии с K-fold. Сравниваем с `CalibratedClassifierCV` из sklearn.

**2.2. Робастная логистическая регрессия**  
Реализуем `BernoulliMixtureClassifier` — EM-алгоритм для смеси двух распределений Бернулли. Проверяем, что модель устойчива к порче таргетов.

**2.3. Байесовский подход**  
Оцениваем апостериорное распределение весов через приближение Лапласа, семплируем разделяющие гиперплоскости.
